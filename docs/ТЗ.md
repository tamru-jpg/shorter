# Техническое задание по разработке веб‑приложения на Flask на тему "Шортер ссылок с анализом переходов"

## 1. Цель проекта

Создать сервис сокращения длинных URL-адресов с подробной аналитикой переходов, где пользователь может генерировать
короткие ссылки и отслеживать статистику посетителей для оценки эффективности распространения контента.

## 2. Целевая аудитория

Маркетологи, SMM-специалисты и владельцы контента,
которым нужно анализировать трафик по ссылкам в соцсетях, email-рассылках и рекламе.

## 3. Основные функции (функциональные требования)

**Функции:**

1. Пользователь может зарегистрироваться и войти в свой аккаунт.
2. Пользователь может сокращать длинные ссылки и получать уникальные короткие URL.
3. Пользователь может просматривать список всех своих коротких ссылок с базовой статистикой.
4. Пользователь может детально анализировать переходы по каждой ссылке (гео, устройство, время).
5. Пользователь может генерировать QR-коды для своих коротких ссылок.

## 4. Структура веб‑приложения (основные страницы)

| Название страницы    | Описание                                  | Доступные функции                                           |
|----------------------|-------------------------------------------|-------------------------------------------------------------|
| Главная              | Стартовая страница с формой сокращения    | Поле для ввода URL, кнопка «Сократить», авторизация         |
| Регистрация          | Форма создания аккаунта                   | Ввод email, пароля, имени; кнопка «Зарегистрироваться»      |
| Логин                | Форма входа в систему                     | Ввод email и пароля; кнопка «Войти»                         |
| Мои ссылки           | Список всех коротких ссылок пользователя  | Просмотр ссылок, просмотр статистики, кнопка «Новая ссылка» |
| Детальная статистика | Детальная статистика по конкретной ссылке | Графики переходов                                           |
| Статистика           | Общая дашборд-статистика аккаунта         | Обзор всех ссылок, суммарная аналитика, графики             |
| QR-код ссылки        | Генерация QR-кода для выбранной ссылки    | Кнопка «Сгенерировать QR», скачивание , предпросмотр        |

## 5. Данные и базы данных

**Сущности и их свойства:**

- Пользователь: id, имя, email, пароль (зашифрованный хэш), дата регистрации.
- Ссылка: id, user_id, оригинальный url, короткий код, дата создания, количество переходов.
- Переход: id, link_id, ip_адрес, страна, город, устройство, браузер, операционная система, дата время.

## 7. Чек‑лист для самопроверки (как понять, что проект готов?)

**Критерии:**

1. Регистрация и авторизация работают корректно с сессиями.
2. Длинные ссылки успешно сокращаются в уникальные короткие коды.
3. Короткие ссылки корректно перенаправляют на оригинальный URL.
4. Все переходы по коротким ссылкам логируются в БД с геоданными.
5. Статистика по переходам отображается в реальном времени с графиками.
6. QR-коды генерируются и скачиваются корректно для всех коротких ссылок.
