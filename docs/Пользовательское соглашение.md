2.1. Для использования сервиса Пользователь обязан пройти регистрацию, указав достоверные данные:

2.3. Пользователь обязуется немедленно уведомить Администрацию о любом несанкционированном доступе к своему аккаунту.

2.4. Администрация вправе удалить аккаунт Пользователя без предварительного уведомления в случае:
- Нарушения условий настоящего Соглашения

## 3. Правила использования сервиса

### 3.1. Пользователь обязуется:

- Не использовать сервис для распространения запрещенной информации
- Не создавать ссылки на вредоносные или фишинговые сайты
- Не пытаться взломать или нарушить работу сервиса
- Не использовать сервис для спама

### 3.2. Пользователю запрещается:

- Создавать ссылки на материалы экстремистского характера
- Распространять порнографический контент
- Использовать сервис для обхода блокировок
- Продавать короткие ссылки третьим лицам без согласия Администрации

### 3.3. Администрация оставляет за собой право:

- Блокировать любую ссылку, нарушающую правила
- Удалять аккаунт Пользователя без объяснения причин

## 5. Конфиденциальность и обработка данных

### 5.1. Какие данные собирает сервис:

| Тип данных | Цель сбора |
|------------|------------|
