# Инструкция по использованию сервиса SHORTENER

## Что такое SHORTENER?

SHORTENER — это сервис сокращения ссылок с аналитикой. Вы можете превратить любую длинную ссылку в короткую, удобную для отправки и отслеживать статистику переходов.

---

## Быстрый старт

### 1. Регистрация

1. Перейдите на главную страницу
2. Нажмите кнопку **"Создать аккаунт"**
3. Заполните поля:
   - **Имя пользователя** (как вас будут видеть другие)
   - **Email** (для входа и уведомлений)
   - **Пароль** (минимум 6 символов)
4. Нажмите **"Зарегистрироваться"**
5. После регистрации войдите в аккаунт

### 2. Создание короткой ссылки

#### Способ 1: Быстрое создание (на главной странице)

1. После входа вы находитесь на главной странице
2. Вставьте длинную ссылку в поле **"Вставьте URL"**
3. Нажмите кнопку **"Сократить"**
4. Готово! Короткая ссылка появится в поле ниже

#### Способ 2: Создание через личный кабинет

1. Перейдите в раздел **"Мои ссылки"**
2. Нажмите **"Создать новую ссылку"**
3. Введите URL и (опционально) название
4. Нажмите **"Создать"**

### 3. Копирование ссылки

- Нажмите кнопку **"Копировать"** рядом с короткой ссылкой
- Ссылка автоматически сохранится в буфер обмена

## Управление ссылками

### Где хранятся мои ссылки?

Все созданные ссылки находятся в разделе **"Мои ссылки"** в меню.

### Что я вижу в таблице ссылок?

| Колонка | Описание |
|---------|----------|
| Короткая ссылка | Ваша короткая ссылка (можно нажать для перехода) |
| Оригинальная ссылка | Длинный URL, который вы сократили |
| Дата создания | Когда была создана ссылка |
| Переходы | Сколько раз перешли по ссылке |
| Действия | Кнопки для управления ссылкой |

### Действия со ссылкой

- **📋 Копировать** — скопировать короткую ссылку
- **📊 Статистика** — посмотреть детальную аналитику
- **📱 QR** — сгенерировать QR-код для ссылки
- **🗑️ Удалить** — удалить ссылку (восстановить нельзя)

## Статистика и аналитика

### Общая статистика (раздел "Статистика")

- Общее количество ссылок
- Общее количество переходов
- Среднее число переходов на ссылку

### Детальная статистика по ссылке

Чтобы открыть, нажмите кнопку **"Статистика"** у нужной ссылки.

#### Что показывается:

| Блок | Что можно узнать |
|------|------------------|
| **Информация о ссылке** | Короткая ссылка, оригинальный URL, дата создания |
| **Динамика переходов** | График переходов по дням (последние 7 дней) |
| **География** | Страны и города, из которых переходили |
| **Устройства** | Desktop / Mobile / Tablet |
| **Браузеры** | Chrome, Firefox, Safari, Edge, Opera |
| **Операционные системы** | Windows, macOS, Linux, Android, iOS |
| **Последние переходы** | Подробный список со временем, IP, Referer |

## QR-коды

### Как сгенерировать QR-код?

1. На странице **"Мои ссылки"** нажмите кнопку **"QR"** у нужной ссылки
2. QR-код откроется в модальном окне

### Как скачать QR-код?

- Нажмите кнопку **"Скачать QR-код"** в модальном окне
- QR-код сохранится в формате PNG

## Часто задаваемые вопросы (FAQ)

### ❓ Сколько ссылок можно создать?

Неограниченное количество.

### ❓ Есть ли ограничение на длину ссылки?

Оригинальная ссылка может быть любой длины. Короткая ссылка всегда имеет 6-10 символов.

### ❓ Как долго хранятся ссылки?

Ссылки хранятся постоянно, пока вы их не удалите.

### ❓ Могу ли я создать свой короткий код?

Да! При создании ссылки вы можете указать свой короткий код вместо автоматического.

### ❓ Видит ли кто-то мою статистику?

Только вы видите статистику по своим ссылкам. Администратор тоже может видеть (только в целях отладки).

### ❓ Что такое "Referer" в статистике?

Referer показывает, с какого сайта пользователь перешел на вашу ссылку. Например, из Telegram, ВКонтакте или с другого сайта.

### ❓ Безопасно ли использование сервиса?

Да. Пароли хранятся в зашифрованном виде. Ваши данные не передаются третьим лицам.
